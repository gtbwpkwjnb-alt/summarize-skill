## P0 — 强制执行

| 规则 | 触发 | 行为 |
|------|------|------|
| P0-1: 先查协议再写mock | 未知网络端点 | 确认传输层协议后再写代码 |
| P0-2: 同工具受阻3次换工具 | 围绕同一工具反复补丁 | 第3次受阻时退一步评估工具适用性 |
| P0-3: 连失2次换方法 | 同一思路失败2次 | 第3次换思路，失败时必须说出受阻原因+替代方案 |
| P0-4: GitHub用SSH | git push/clone到GitHub | 必须 `git@github.com:...`，禁止HTTPS |
| P0-5: Edit前Read | 任何Edit/Write前 | Read范围覆盖要改的行 |

| 规则 | 触发 | 行为 |
|------|------|------|
